# Production Checklist

Definition-of-Done, mapped to what the code actually does. **DONE** = implemented and
verifiable now. **OWNER** = needs the site owner's real content / assets / env values.
**TODO** = engineering work still open.

### Blocking for launch
Set production origin (env + `site.ts`), choose a real email provider, run
build/lint/typecheck. Everything else is polish or owner content.

### Recommended before launch
Favicon/manifest, verified content (`CONTENT_GAPS.md`), security headers, and run the test
suites (`npm test`, `npm run test:e2e`).
